Anonymous-pi function, where the name is unused and therefore not written Note the implicit expression argument, as otherwise sugar = (app >> arrow >> sugar) | app If this were not the case, all expressions are parsed as anonymous pis and will fail at the final arrow This is an exponential slowdown S. :: π(_:S).S
